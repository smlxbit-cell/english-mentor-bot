# Spirit — эмоции и события бота

Дух **Spirit** сопровождает ученика короткими клипами. Формат в чате всегда один:

1. **Круглое видео** (video note, ~200px) — если клип зарегистрирован  
2. **Текст** — приветствие, вопрос, кнопки  

Не каждое сообщение бота сопровождается эмоцией. Только **ключевые повороты** сценария.

---

## Папки с файлами

```
media/spirit/
  emotions/          ← исходники (любой размер, 3–6 сек)
    greeting.mp4
    lets_go.mp4
    success.mp4
    think.mp4
    explain.mp4
    support.mp4
    surprise.mp4
    applause.mp4
    complete.mp4
    pause.mp4
    goodbye.mp4
    sleep.mp4        ← НОВЫЙ: давно не заходил
  notes/             ← опционально: готовые квадраты 1:1 (иначе sync сам кропнет)
  scenes/            ← картинки эпизодов (не круг)
```

После загрузки файлов:

```bash
python manage.py sync_spirit_media --force
python manage.py spirit_status
```

Sync создаёт круглые версии автоматически и **не оставляет** огромные GIF в чате.

## Список эмоций (12 файлов)

| Ключ файла | Настроение | Когда показываем |
|------------|------------|------------------|
| `greeting` | приветливый, машет | Обычный `/start`, возврат в тот же день |
| `sleep` | сонный, зевает, «просыпается» | Не заходил **≥ 7 дней** |
| `lets_go` | энергия, «поехали» | Старт **нового** эпизода (шаг 0) |
| `pause` | мягкое «эй, вернись» | Незаконченный урок; **напоминание** по расписанию |
| `success` | радость | Правильный ответ (не каждый подряд — см. ниже) |
| `think` | задумался | Неправильный ответ |
| `explain` | учительский | Шаг **грамматики** в уроке |
| `support` | поддержка | Старт **тренировки слов** |
| `surprise` | вау | **Клиффхэнгер** в конце эпизода |
| `complete` | довольный | Эпизод **пройден** |
| `applause` | аплодисменты | Level-up или особо сильный финиш |
| `goodbye` | машет на прощание | Редко: явный уход («до завтра») — *запас* |

### Опционально позже (не обязательно сейчас)

| Ключ | Идея |
|------|------|
| `curious` | Перед диагностикой уровня |
| `listen` | Перед заданием на аудирование |
| `proud` | Длинный стрик (7+ дней) |

## События бота → эмоция

Код: `telegram_app/bot/mentor.py` → `SPIRIT_MOMENTS`.

### Уровень A — всегда круг + текст

| Событие (`moment`) | Эмоция | Триггер в боте |
|--------------------|--------|----------------|
| `welcome_back` | greeting | `/start`, был недавно |
| `long_absence` | sleep | `/start`, нет **≥ 7 дней** |
| `lesson_resume` | pause | `/start`, есть незаконченный урок |
| `lesson_start` | lets_go | «▶️ Продолжить», шаг 0 |
| `lesson_complete` | complete | Финиш эпизода |
| `lesson_complete_big` | applause | Level-up в том же финале |
| `cliffhanger` | surprise | Шаг cliffhanger в сценарии |
| `reminder` | pause | Cron `send_reminders` |

### Уровень B — круг + текст на важных шагах

| Событие | Эмоция | Триггер |
|---------|--------|---------|
| `grammar_note` | explain | Шаг grammar |
| `word_review` | support | Старт повторения слов |
| `answer_wrong` | think | Неверный ответ в упражнении |

### Уровень C — лёгкие реакции (не спам)

| Событие | Эмоция | Правило |
|---------|--------|---------|
| `answer_correct` | success | После ошибки **или** каждый ~3-й правильный ответ |

### Без эмоции

- Обычные шаги story / vocabulary / matching  
- Меню «Профиль», «Правила», «Словарь»  
- План дня (только текст «Глава дня»)  
- Подсказки, озвучка, мелкие подтверждения  

### Сцены эпизода

Картинки `scenes/ep01_cafe.jpg` — **фото**, не круг. Показываются на шагах hook/story по `scene_key` в curriculum.

## Тексты рядом с эмоциями (черновик)

### `welcome_back` + greeting
```
С возвращением, {имя}! 👋
Твой уровень: {CEFR}.
Выбери, чем займёмся 👇
```

### `long_absence` + sleep (≥ 7 дней)
```
{name}, скучали! Тебя не было {N дней}.
Уровень: {CEFR}.
Начнём с плана дня?
```

### `lesson_resume` + pause (бросила урок на полпути)
```
{name}, ты не закончила эпизод «{title}».
Остановились на шаге {k}. Продолжим?
[▶️ Продолжить урок]
```

Тот же `pause` при возврате в урок из плана (шаг > 0):
```
Продолжаем «{title}» — шаг {k} из {total}.
```

### `lesson_start` + lets_go
```
▶️ Эпизод: {title}
(далее первый шаг сценария)
```

### `reminder` + pause
```
Привет, {имя}! Пора к английскому — план на сегодня ждёт.
[📚 Открыть план дня]
```

## Два разных сценария «давно не было»

| | Когда | Как |
|---|--------|-----|
| **Возврат** (`/start`) | Пользователь **сам зашёл** после 7+ дней | `sleep` + «Скучали! Тебя не было N дней…» |
| **Проактивный пинг** | Пользователь **не заходит** 7+ дней | Cron `send_inactive_nudge` → `sleep` + совет + цитата |

Пинг отправляется **один раз** за период отсутствия (не каждый день). После нового визита счётчик сбрасывается — если снова пропадёт на 7 дней, придёт новое сообщение.

### Текст проактивного пинга (пример)
```
Мария, мы скучаем! 🌙
Тебя не было 9 дней.

💡 Не бойся ошибок — они показывают, где расти…

✨ «Consistency beats intensity.»
🇬🇧 Small daily habits beat rare marathons.

Spirit ждёт — загляни хоть на 10 минут?
[📚 Вернуться к учёбе]
```

Советы и цитаты — пул в `telegram_app/inactive_tips.py` (15 советов, ротация по user_id).

## Cron

| Команда | Расписание | Эмоция |
|---------|------------|--------|
| `send_reminders` | Каждый час | `pause` — план на сегодня |
| `send_inactive_nudge` | 1× в день (напр. 10:00) | `sleep` — не заходил 7+ дней |

```bash
python manage.py send_inactive_nudge --dry-run   # кого затронет
python manage.py send_inactive_nudge             # отправить
```

## Тест сценарий

1. `/start` → круг greeting → текст → меню  
2. 📚 Учиться → ▶️ Продолжить → круг lets_go → intro  
3. Неверный ответ → круг think → фидбек  
4. Грамматика → круг explain → правило  
5. Конец эпизода → круг complete → XP  
6. Клиффхэнгер → круг surprise → текст  
7. Слова → круг support → задание  
8. *(симуляция)* `send_reminders --hour …` → круг pause → текст
